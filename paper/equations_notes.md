# Equations Notes

The LaTeX equations for the FastBPE paper are collected in `paper/equations.tex`.

Use that file for:

- throughput definitions
- latency definitions
- exact match rate
- token count difference
- cache hit rate
- speedup
- memory efficiency
- the correctness-constrained optimization statement

If you cite equations in the main paper, copy the relevant blocks into the methodology or appendix sections rather than duplicating ad hoc definitions.
